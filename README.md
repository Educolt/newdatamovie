## 🗂 Como instalar o projeto

```bash
  # Clonar o repositório
  $ git clone https://github.com/Educolt/newdatamovie

  # Entrar no diretório
  $ cd newdatamovie

  # Instalar as dependencias
  $ yarn

  # Inicia o projeto
  $ expo start

```
